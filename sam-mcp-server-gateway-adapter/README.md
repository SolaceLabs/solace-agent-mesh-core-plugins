# SamMcpServerGatewayAdapter SAM Plugin

A gateway adapter to make SAM accessable as a MCP server

This is a plugin for Solace Agent Mesh (SAM).

## Installation

Once the plugin is installed (e.g., from PyPI or a local wheel file):
```bash
sam plugin add <your-new-component-name> --plugin sam-mcp-server-gateway-adapter
```
